# PCRPractice

## Overview
PCRPractice is a Patient Care Report (PCR) training module. It features a narrative editor, SOAP/CHART logic, and scoring engines to help students master documentation.

## Original Paths
- `src/features/pcr-trainer`
- `src/data/tools/pcr-documentation.ts`
- `src/data/enhanced-pcr-practice.ts`

## Dependencies
- React
- TypeScript
- Tailwind CSS
- Lucide React (icons)
